Через запрос это будет выглядеть так .

```mysql
INSERT INTO `first_bd`.`posts` (`id`, `title`, `content`, `category`, `like`, `is_published`) VALUES ('3', 'cat title', 'cat content', 'cats', '200', '1');
```
