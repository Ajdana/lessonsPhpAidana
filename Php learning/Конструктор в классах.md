**Конструктор** — это специальный метод `__construct`, который вызывается **при создании объекта**.

```php
class Person {
    public $name;
    public $age;
    public $hobby;

    public function __construct($name, $age, $hobby) {
        echo 'CONSTRUCTOR!!!!!';
        $this->name = $name;
        $this->age = $age;
        $this->hobby = $hobby;
    }
}

$person = new Person('Bob', 20, 'swimming');

echo $person->name; // Bob
```
